# Support

## Getting Help

There are several ways to get help with cpp-quick-starter:

### Documentation

- **README**: Start with the [README.md](README.md) for quick start instructions
- **Guides**: Check the [docs/guides/](docs/guides/) directory for detailed guides
- **API Documentation**: Build and view the Doxygen documentation

### Community Support

- **GitHub Discussions**: Ask questions and share ideas in [Discussions](https://github.com/AstroAir/cpp-quick-starter/discussions)
- **GitHub Issues**: Report bugs or request features via [Issues](https://github.com/AstroAir/cpp-quick-starter/issues)

### Issue Guidelines

Before opening an issue:

1. Search existing issues to avoid duplicates
2. Use the appropriate issue template
3. Provide as much detail as possible:
   - Operating system and version
   - Compiler and version
   - CMake/xmake version
   - Steps to reproduce
   - Expected vs actual behavior

### Response Time

This is an open-source project maintained by volunteers. Please be patient:

- **Bug reports**: We aim to triage within 1 week
- **Feature requests**: Reviewed periodically
- **Pull requests**: Reviewed as time permits

## Commercial Support

For commercial support or consulting, please contact the maintainers directly.
